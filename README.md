# Galaxy Shooter 


A simple 2D Space Shooter game developed in Java using Swing and Object-Oriented Programming (OOP) concepts.

Description:

Galaxy Shooter is an arcade-style shooting game where the player controls a spaceship, shoots enemies, and tries to survive as long as possible while achieving a high score.
This project was developed as part of an Object-Oriented Programming (OOP) course project.

Features:

1)Player movement using arrow keys
2)Shooting system
3)Enemy spawning
4)Collision detection
5)Score tracking
6)Health system
7)Game Over screen
8)Custom graphics and backgrounds

How to Run:

1. Clone the repository:
   git clone <repository-url>
2. Open the project in IntelliJ IDEA or any Java IDE.
3. Make sure JDK 8 or later is installed.
4. Run:
   Main.java

Controls:

| Key   | Action     |
| ----- | ---------- |
| ↑     | Move Up    |
| ↓     | Move Down  |
| ←     | Move Left  |
| →     | Move Right |
| Space | Shoot      |
